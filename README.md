# Baytkay GitHub Pages Landing Page

This is a static landing page rebuilt from the previous Strikingly landing page structure and optimized for ad traffic.

## Files

- `index.html` — landing page content and structure
- `style.css` — all styling

## How to publish on GitHub Pages

1. Create a new public GitHub repository, for example `baytkay-london-funding`.
2. Upload `index.html` and `style.css` into the repository root.
3. Go to **Settings**.
4. Click **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select `main` branch and `/root` folder.
7. Save.
8. Your page will publish at a URL like `https://yourusername.github.io/baytkay-london-funding/`.

## Stronger setup

For ads, connect a custom subdomain such as:

- `study.easytravel.ng`
- `apply.easytravel.ng`
- `london.easytravel.ng`

A custom domain will look more trustworthy than a raw GitHub URL.

Replace or confirm the WhatsApp CTA number, pricing, processing cost and course details before sending paid traffic.
